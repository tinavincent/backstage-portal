# Product Description

## Vision

Create a personal portal that provides a single entry point to all vibecoding projects, AI agents, documentation and shared development resources.

The portal should help Tina quickly discover, understand and continue work across all projects from one place.

It complements existing tools rather than replacing them.

---

## Problem

My vibecoding projects, AI agents, documentation and development resources are spread across multiple GitHub repositories and local folders.

I lack a central catalog where I can quickly:

- See which projects exist.
- Understand each project's purpose and current status.
- Find documentation.
- Find AI agents and understand their responsibilities.
- Find skills, prompts and playbooks.
- Find MCP servers and shared components.
- Continue where I left off.

The portal should collect and visualize information from existing tools instead of replacing them.

---

## Users

### Primary User

Tina

The portal is designed as a personal workspace for managing and evolving vibecoding projects.

The goal is to provide a single place to discover, understand and continue work across projects, AI agents and documentation.

Support for additional users may be added later but is out of scope for the MVP.

---

## Value Proposition

Backstage Portal gives Tina one place to discover, understand and continue work across all vibecoding projects.

Instead of searching across repositories, folders and documents, the portal provides a single entry point with project overviews, documentation and links to the underlying resources.

The portal complements existing tools such as GitHub, Cursor and Claude Code rather than replacing them.

---

## MVP

The first version should allow Tina to:

- Browse all vibecoding projects.
- Open each project's documentation.
- View the product description.
- View the project principles.
- Navigate to the project's GitHub repository.
- Discover projects from a single portal.

### Out of scope

The following ideas are intentionally postponed until after the MVP:

- AI agent catalog
- GitHub Projects integration
- "What should I work on tonight?" recommendations
- Project milestone timeline
- Project story visualization
- Custom Backstage plugins
- Knowledge graph between projects

---

## Long-term Vision

Over time, the portal should evolve into a personal product development workspace.

Potential future capabilities include:

- AI agent catalog
- Skills and prompt catalog
- MCP server catalog
- Shared component catalog
- Automatic project discovery
- Project milestone timeline
- Project Story visualization
- Personalized work recommendations
- Knowledge graph showing relationships between projects
- Portfolio view of completed projects
